# Sistema de Login em Python

Projeto simples de um sistema de login com cadastro de usuários, desenvolvido em Python.

---

##  Sobre o projeto
Este projeto simula um sistema básico de autenticação, permitindo que usuários se cadastrem e realizem login com validação de dados.

---

##  Funcionalidades
✔ Cadastro de usuários  
✔ Validação de login  
✔ Verificação de usuário existente  
✔ Menu interativo no terminal  

---

## Como executar

1. Clone o repositório:
```bash
git clone https://github.com/gikasouza/sistema-login-python.git
```

2. Acesse a pasta:
```bash
cd sistema-login-python
```

3. Execute o programa:
```bash
python login.py
```
